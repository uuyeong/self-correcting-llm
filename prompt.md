## 📄 파일 2: Overleaf LaTeX 템플릿 사용법

보고서 작성을 위한 Overleaf 사용 방법을 단계별로 안내합니다.

### 사용 순서

1. Overleaf에서 **"New project"** 클릭
2. **"Upload project"** 선택
3. 제공된 **`template.zip`** 파일 업로드
4. 프로젝트가 생성되면 **`main.tex`** 파일에서 논문 작성

### 파일 구조

| 파일 | 역할 |
| --- | --- |
| `main.tex` | 실제 논문 내용 작성 |
| `egbib.bib` | 참고문헌 BibTeX 정보 입력 |
| `cvpr.sty` | 스타일 설정 (수정 불필요) |
| `ieee_fullname.bst` | 참고문헌 형식 (수정 불필요) |

### 참고문헌 사용법

- `egbib.bib`에 BibTeX 형식으로 문헌 정보 추가
- `main.tex`에서 `\cite{Authors14}` 형식으로 인용
